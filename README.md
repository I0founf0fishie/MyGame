#  Sheep and Dream — Овечка и Сон

Прыжковая аркада с механикой облаков, гроз и сахарных бонусов.  
Управляйте овечкой, избегайте опасностей и набирайте очки, пока скорость растёт.

##  Геймплей

- Овечка **бежит вправо автоматически**.
- Перепрыгивайте между облаками — упадёте, проиграете жизнь.
-  **Обычные облака** — безопасны.
-  **Грозовые облака** — забирают жизнь.
-  **Сахарные облака** — добавляют жизнь (максимум 3).
-  **Звёзды** — опасны, отнимают жизнь.
- Игра **ускоряется** с ростом счёта.

##  Управление

| Действие               | Клавиша / Мышь              |
|------------------------|-----------------------------|
| Прыжок                 | `Space` или ЛКМ             |
| Пауза                  | `P` или `Esc`               |
| Перезапуск (после конца) | `R`                       |
| Выход                  | закрыть окно                |

##  Установка и запуск

### 1. Установите Python 3.7+

### 2. Установите Pygame

```bash
pip install pygame
```

### 3. Скачайте репозиторий

```bash
git clone https://github.com/I0founf0fishie/MyGame.git
cd MyGame
```

### 4. Запустите игру

```bash
python sheep-and-dream.py
```

Если ассетов нет, игра отобразит цветные фигуры (но выдаст предупреждения в консоль).
